***********************************************************************************************************************************************************************************
RegisterUser IS A PROJECT TO SELECT, INSERT, UPDATE, DELETE ACCORDING TO USER CONTROL ON WEB APP GUI.
***********************************************************************************************************************************************************************************

1. HOME = Display Welcome page to guide user to use the GUI features

2. PRIVACY = To remind any developers on project confidentiality and copyright

3. Users = To direct webpage to view all data from 'users' table
Sub-features:
- Edit = Update data according to user's will and synchronize data on 'users' table on 'registerdata' database
- Delete = Delete record(row) according to user's will and synchronize data on 'users' table on 'registerdata' database
